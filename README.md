# FUTURE_ML_02
# Customer Churn Prediction

## Overview
This project predicts customer churn using machine learning and visualizes insights in Power BI.

## Tools Used
- Python (Pandas, Scikit-learn, XGBoost)
- Power BI

## Files
- churn_dashboard.pbix – Power BI dashboard
- churn_powerbi.csv – Dataset with churn probabilities
- churn_model.py – ML model code

## Insights
- Month-to-month customers show highest churn
- High monthly charges increase churn probability
- Low tenure customers are high risk
